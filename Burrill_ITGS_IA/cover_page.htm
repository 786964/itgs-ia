Directions to access product

In Firefox, click on the xpi file to install the extension bundle. Click "Add" when prompted for permission.

Download and extract the mock server bundle for your platform and run (in a terminal window) the executable contained in the extracted folder (ezymock or ezymock.exe).

Go to the address printed out to the terminal (Usually 127.0.0.1:5000).

Select the Patients tab on the mock site and choose a patient from the left sidebar.

Click on the extension icon (Cat looking at a monitor) to access the primary interface.

To test in Chrome, you will need to install the extension manually, see Product/troubleshooting.pdf.

 If the product does not load, please go to the Product folder to find the files (you can drag and drop the xpi into Firefox), or take a look at Product/troubleshooting.pdf for running from source.
